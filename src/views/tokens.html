Tokens

In order to facilitate cross-platform development, this tool utilizes 'design tokens'. The tokens are defined and translated into multiple languages using Dragoman.

Using tokens as a variable within another token requires the % prefix. The percent sign is converted to the proper character during translation into the various languages.
